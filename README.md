# VEXIS-Chat

This is a web interface that allows you to intuitively issue commands to AI agents directly from your browser.
It works in conjunction with [VEXIS-CLI-REMOTE](https://github.com/AInohogosya/VEXIS-CLI-REMOTE).

## 🌐 Runtime Environment (URL)

No installation is required; you can start using it right now via the link below.
**[https://ainohogosya.github.io/VEXIS-Chat/](https://ainohogosya.github.io/VEXIS-Chat/)**

---

## 🚀 How to Use

1.  **Start the Backend**
    Clone [VEXIS-CLI-REMOTE](https://github.com/AInohogosya/VEXIS-CLI-REMOTE)—the main component that receives instructions—complete the setup, and start it.
2.  **Access the Website**
    Open [VEXIS-Chat](https://ainohogosya.github.io/VEXIS-Chat/) in your browser.
3.  **Issuing Instructions to the Agent**
    Send instructions to the AI agent using the input form on the screen.

---

## 🛠 About the Project

This app is being developed as part of the proprietary AI agent ecosystem "VEXIS."

* **Status:** In development (Pre-alpha)
* **Purpose:** To convert CLI operations into a GUI and make agent operations in remote environments more convenient.
* **Integration:** [VEXIS-CLI-REMOTE](https://github.com/AInohogosya/VEXIS-CLI-REMOTE)

## 👨‍💻 For Developers (Local Execution)

Follow the steps below if you wish to customize the repository or build it locally.

```bash
# Clone the repository
git clone [https://github.com/AInohogosya/VEXIS-Chat.git](https://github.com/AInohogosya/VEXIS-Chat.git)

# Change to the directory
cd VEXIS-Chat

# Install dependencies
npm install

# Start the development server
npm run dev
```
